# WaveX-Watercraft-Sales-and-HR-Analytics_Power-BI

## Overview

This project analyzes sales and HR data for an imaginary company, WaveX Watercraft, using Power BI. The objective is to clean, transform, and model the data to generate meaningful insights and build an interactive dashboard with forecasting capabilities.

---

## Tools and Technologies Used

* Power BI Desktop
* Power BI Service (Cloud Publishing)
* Power Query (M Language)
* DAX (Data Analysis Expressions)

---

## Dataset Description

### Fact Tables

* Sales
* HR

### Lookup Tables

* Distributor
* Category
* Product Name

---

## Data Preparation (Power Query)

* Excluded records from the year 2018
* Removed sales data for the discontinued product WR3
* Performed data cleaning and transformation using Power Query
* Applied basic transformations using M language

---

## Data Modeling

* Built relationships between fact and lookup tables
* Designed a structured data model for efficient analysis
* Ensured proper cardinality and filtering behavior

---

## Key Features

### KPI Cards

* Created KPI cards to highlight key business metrics such as Total Sales and performance indicators
* Enabled quick, high-level insights

### Interactive Visualizations

* Developed charts to analyze sales trends, category performance, and distributor insights
* Integrated HR data for additional analysis

### Slicers (Filters)

* Implemented slicers to allow dynamic filtering across visuals
* Enabled users to explore data interactively

### Navigation Buttons

* Added navigation buttons to switch between report pages
* Created a seamless multi-page user experience

### Forecasting

* Implemented a two-year sales forecast using Power BI analytics features
* Enabled forward-looking insights for decision-making

---
